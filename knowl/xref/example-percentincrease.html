Example 3.1.11. Percentage Increase.

In the news, you hear “tuition is expected to increase by 7% next year.” If tuition this year was $970 per year, what will it be next year?
Solution.
In this situation, the current tuition, $970, is the whole, the percent is 7%, and what we want to know (almost) is again the part. Converting 7% to the decimal 0.07 and using the same formula as above gives:
\begin{align*} \text{PART} \amp = \text{PERCENT} \cdot \text{WHOLE}\\ \amp = 0.07 \cdot $970\\ \amp = $67.90 \end{align*}
In Example 3.1.7, we were done at this point. However, this question does not ask how much is added on to the current tuition, it asks for us to compute the new tuition. So, we need to add this increase to the current amount:
\begin{equation*} \text{New Tuituion} = $970.00 + $67.90 = $1037.90\text{.} \end{equation*}
Notice again that the final answer seems reasonable (is higher than the previous year) and that the units make sense.
Alternatively, if you don’t like the two step approach (or if it just makes more sense this way), you can reconsider the value we use for the percent. Since 7% is the amount increased, that is in addition to the 100% of the tuition from the previous year. So, we can still use $970 for the whole, but change the percent to 107% = 1.07, and the part will now be the next year’s tuition.
\begin{align*} \text{PART} \amp = \text{PERCENT} \cdot \text{WHOLE}\\ \amp = 1.07 \cdot $970\\ \amp = $1037.90 \end{align*}
