    Section 2.6 Logarithms and Solving for the Exponent

    There is a lot one could say and learn about logarithms. We will aim to keep this section as straightforward as possible, as we only need logs as a tool for solving equations that involve exponentials.
    🔗
    While there is a whole family of logarithms with different bases, we will simplify our study by focusing on the common log, which is based on the exponential \(10^x\text{.}\)
    🔗

    Definition 2.6.1. Common Logarithm.

    The common logarithm, written \(\log(x)\text{,}\) undoes the exponential \(10^x\text{.}\) This means that \(\log\left(10^x\right) = x\text{,}\) and likewise \(10^{\log(x)} = x\text{.}\) This also means the statement \(10^a = b\) is equivalent to the statement \(\log(b) = a\text{.}\)
    🔗
    🔗

    Good Brain Habit 2.6.2.

    Words are important. When we read the set of symbols \(\log(x)\) out loud (or maybe even more importantly, when we’re just reading to ourselves), we should say “log OF \(x\text{,}\)” or even more accurately, “log base 10 OF \(x\text{.}\)” The focus here is on “OF” so that we do not confuse the parentheses as multiplication; \(\log(x)\) DOES NOT mean “log times x.” Another approach to help remember that this is not multiplication, is that “\(\log\)” by itsels means nothing, just like \(\sqrt{\hspace{8pt}}\) or \(+\text{.}\)
    🔗
    🔗

    Example 2.6.3.

    Evaluate \(\log(100)\) and \(\log\left(\frac{1}{100}\right)\text{.}\)
    🔗
    Solution.
    \(\log(100)\) can be read as “What power do we raise 10 to in order to get 100?” We probably recoginize that \(10^2 = 100\text{,}\) so \(\log(100) = 2\text{.}\)
    🔗
    Recall from Section 2.4, that \(x^{-n} = \frac{1}{x^n}\text{,}\) so
    \begin{equation*} \log\left(\frac{1}{100}\right) = \log\left(\frac{1}{10^2}\right) = \log\left(10^{-2}\right) = -2. \end{equation*}
    🔗
    🔗
    🔗

    Check for Understanding 2.6.4.

    Evaluate the following expressions:
    1. \(\displaystyle \log(10000)\)
      🔗
      🔗
    2. \(\displaystyle \log(1)\)
      🔗
      🔗
    3. \(\displaystyle \log\left(\frac{1}{1000}\right)\)
      🔗
      🔗
    🔗
    Solution.
    1. \(\displaystyle \log(10000) = \log\left(10^4\right) = 4.\)
      🔗
      🔗
    2. \(\displaystyle \log(1) = \log\left(10^0\right) = 0.\)
      🔗
      🔗
    3. \(\displaystyle \log\left(\frac{1}{1000}\right) = \log\left(\frac{1}{10^3}\right) = \log\left(10^{-3}\right) = -3.\)
      🔗
      🔗
    🔗
    🔗
    🔗

    Good Brain Habit 2.6.5.

    It is worth noting that in the example and practice above, we used the word “evaluate” rather than “solve.” This is an important distinction; we use the word “solve” when we want to determine the value of a variable in an equation, as below.
    🔗
    🔗

    Example 2.6.6.

    Solve the following equations for the variable \(x\text{:}\)
    1. \(\displaystyle 10^x = 1000\)
      🔗
      🔗
    2. \(\displaystyle 10^x = 3\)
      🔗
      🔗
    3. \(\displaystyle 2\left(10^x\right) = 8\)
      🔗
      🔗
    🔗
    Solution.
    1. Taking the log of both sides gives \(\log\left(10^x\right) = \log(1000)\text{.}\) Since the log undoes the exponential, \(\log\left(10^x\right) = x\text{.}\) Similarly, \(\log(1000) = \log\left(10^3\right) = 3\text{.}\) The equation simplifies then to \(x = 3\text{.}\)
      🔗
      🔗
    2. Taking the log of both sides gives \(\log\left(10^x\right) = \log(3)\text{.}\) On the left side, \(\log\left(10^x\right) = x\text{,}\) so \(x = \log(3) \approx 0.477\text{.}\) We evaluate \(\log(3)\) using a calculator or computer.
      🔗
      🔗
    3. Here we would first want to isolate the exponential by dividing both sides of the equation by 2, giving \(10^x = 4\text{.}\) Now we can take the log of both sides, giving \(\log\left(10^x\right) = \log(4)\text{,}\) which simplifies to \(x = \log(4) \approx 0.602\text{.}\)
      🔗
      🔗
    🔗
    🔗
    The approach in Example 2.6.6 allows us to solve exponential equations with powers of 10. But, as we’ll see in Section 4.3, we’d like to be able to solve equations like \(2 = 1.03^t\text{.}\) For that we need the exponent property for logs.
    🔗

    Fact 2.6.7. Exponent Property of Logarithms.

    For a number \(x\) and any power \(y\text{,}\)
    \begin{equation*} \log\left(x^y\right) = y\log(x). \end{equation*}
    🔗
    🔗
    We can combine this fact with the approaches used in Example 2.6.6 to solve the desired type of equations.
    🔗

    Procedure 2.6.8. Solving Exponential Equations with Logarithms.

    This process is used to solve exponential equations, where the unknown variable appears as the exponent.
    1. Isolate the exponential. In other words, get the number to the unknown power alone on one side of the equation.
      🔗
      🔗
    2. Take the log of both sides of the equation.
      🔗
      🔗
    3. Use the exponent property of logs. Rewrite the side of the equation with the log of the number to the unknown variable as that variable times the log of the number.
      🔗
      🔗
    4. Solve for the variable. This will involve dividing by the log of the base, and maybe other steps as we used in solving linear equations or those involving roots or powers.
      🔗
      🔗
    🔗
    🔗
    It is important to note that Procedure 2.6.8 should be used for exponential equations, where the unknown variable is in the exponent. When the exponents are numbers and unknown appears in the base, that is a power equation and we use roots to solve. See Section 2.5.
    🔗

    Example 2.6.9.

    Solve the equation \(10 = 5\left(1.03^t\right)\) for \(t\text{.}\)
    🔗
    Solution.
    1. First, we isolate the exponential, \(1.03^t\text{,}\) by dividing both sides by 5:
      \begin{align*} \frac{10}{5} \amp = \frac{5\left(1.03^t\right)}{5}\\ 2 \amp = 1.03^t. \end{align*}
      This step will often just involve division as it does here. In more complicated examples, you may simplify things for your brain by replacing the exponential with it’s own new variable, and solving for that in this first step.
      🔗
      🔗
    2. Then, we take the log of both sides:
      \begin{equation*} \log\left(2\right) = \log\left(1.03^t\right). \end{equation*}
      This step is very simply just writing “log” on both sides, but be sure to include everything on each side inside the parentheses.
      🔗
      🔗
    3. Next, we use the exponent property of logs to “bring down” the \(t\) to be outside of the log:
      \begin{equation*} \log(2) = t\log(1.03). \end{equation*}
      Sometimes the exponent is more complicated than just the variable alone. Still the entire expression should be “brought down” and parentheses included so that it is all multiplied by the log.
      🔗
      🔗
    4. Lastly, we regonize that both expressions \(\log(2)\) and \(\log(1.03)\) are just numbers and we can divide both sides by \(\log(1.03)\) to isolate \(t\text{.}\)
      \begin{equation*} \frac{\log(2)}{\log(1.03)} = \frac{t\log(1.03)}{\log(1.03)} = t. \end{equation*}
      Like the first step, isolating the variable here may be more complicated, but involves the same tools and techniques used for other types of equations.
      🔗
      🔗
    We can plug the resulting fraction into a calculator or computer to obtain
    \begin{equation*} t = \frac{\log(2)}{\log(1.03)} \approx 23.45. \end{equation*}
    🔗
    🔗
    🔗
