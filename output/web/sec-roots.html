    Section 2.5 Roots and Solving for the Base

    We saw in Fact 2.4.4 that we can view a square root, or cube root, or any root as just a particular type of exponent, and in fact, that’s often how we enter them into a calculator when we can’t compute them in our heads.
    🔗
    Sometimes we have information about the root of a value, but we want the value itself. Other times, we’ll have information about the value after it has been raised to a power, but want information about the original value. Because terms like “square” and “cube” get thrown into the mix when we’re talking about both the second or third power OR the second or third root... the language can get confusing.
    🔗
    We’ve learned and practiced how to solve linear equations in Section 2.3, now we want to be able to solve equations like \(\sqrt[3]{x}=8\) or \(r^2=42\text{.}\)
    🔗

    Example 2.5.1.

    Solve \(\sqrt[3]{x}=8\) for \(x\text{.}\)
    🔗
    Solution.
    You might read this equation as, “What number cube rooted equals 8?” This can also be thought of as, “8 cubed is what number?” To solve this, we can cube both sides of the equation because cube roots and cubing are inverse functions, they “undo” each other.
    🔗
    \begin{align*} \sqrt[3]{x} \amp = 8\\ \left(\sqrt[3]{x}\right)^3 \amp = \left(8\right)^3\\ x \amp = 8 \times 8 \times 8\\ x \amp = 512 \end{align*}
    🔗
    🔗
    🔗

    Example 2.5.2.

    Solve \(r^2=42\) for \(r\text{.}\)
    🔗
    Solution.
    The equation reads, “What number squared is 42?” This can also be thought of as, “The square root of 42 is what number?” To solve this, we can take the square root of both sides of the equation, again because square roots and squaring are inverse operations.
    🔗
    \begin{align*} r^2\amp = 42\\ \sqrt{r^2} \amp = \sqrt{42} \approx 6.4807 \end{align*}
    Note that \(6.4807 \times 6.4807\) is 42, but \(-6.4807 \times -6.4807 = 42\) as well. So \(r\) is really \(\pm 6.4807\text{.}\)
    🔗
    🔗
    🔗

    Good Brain Habit 2.5.3.

    We can (and should!) always check our answer(s) when solving equations; simply go back to the start and plug in the value(s) and see if it works out.
    🔗
    🔗

    Fact 2.5.4. Even Powers and Roots.

    Raising a negative number to an even power results in a positive value; \((-2)^4 = (-2)(-2)(-2)(-2) = (4)(4) = 16\text{.}\) So, an equation like \(x^n = -1234\) where \(n\) is even will have no solution. This also means, that if we input a negative value into an even root, \(\sqrt[n]{-1234}\) like we would to solve the above equation, most calculators will produce and error. As we see in Example 2.5.2, when determining the solution to an equation with an even power, we will need to consider negative values as well. A calculator will always produce just the positive answer when you input \(\sqrt[n]{\#}\text{.}\)
    🔗
    🔗

    Fact 2.5.5. Odd Powers and Roots.

    Raising a negative number to an odd power results in a negative value; \((-2)^5 = (-2)(-2)(-2)(-2)(-2) = (4)(4)(-2) = -32\text{.}\) So, an equation like \(x^m = -1234\) where \(m\) is odd will have a unique solution; what we get when we input \(\sqrt[m]{-1234}\) into a calculator.
    🔗
    🔗

    Check for Understanding 2.5.6.

    A number cubed is -27, what number (or numbers) is it?
    🔗
    A number squared is -64, what number (or numbers) is it?
    🔗
    A number raised to the fourth power is 81, what number (or numbers) is it?
    🔗
    Solution.
    “A number cubed is -27,” translates to \(x^3 = -27\text{.}\) We can solve this by taking the cube root of both sides:
    \begin{align*} x^3 \amp -27\\ \sqrt[3]{x^3} \amp = \sqrt[3]{-27}\\ x \amp = -3 \end{align*}
    🔗
    “A number squared is -64,” translates to \(x^2 = -64\text{.}\) This will have no solution, since any number raised to an even power is positive, Fact 2.5.4.
    🔗
    “A number raised to the fourth power is 81,” translates to \(x^4 = 81\text{.}\) We can solve this in one step by taking the fourth root of both sides, which is easy with a calculator. Most of these types of equations require a calculator to solve, but this one has fairly friendly numbers.
    🔗
    We might recognize 81 as a “perfect square”, that is \(81 = 9^2\text{.}\) So,
    \begin{align*} x^4 \amp = 81\\ \sqrt{x^4} \amp = \sqrt{81}\\ x^2 \amp = 9\\ \sqrt{x^2} \amp = \sqrt{9}\\ x \amp = 3 \end{align*}
    Since this involves an even power, we consider if a negative might also be a solution:
    \begin{equation*} (-3)^4 = (-3)(-3)(-3)(-3) = (9)(9) = 81. \end{equation*}
    So a number raised to the fourth power that equals 81 could be \(\pm 3\text{.}\)
    🔗
    🔗
    🔗
    In some cases, we’ll need even more steps to solve an equation involving roots or powers. Before we attempt to undo a root with a power or vice versa, we need to make sure the we’ve isolated that term.
    🔗

    Example 2.5.7.

    Solve \(2\sqrt{x} + 1 = 13\) for \(x\text{.}\)
    🔗
    Solution.
    We first aim to isolate the \(\sqrt{x}\) term on one side:
    \begin{align*} 2\sqrt{x} + 1 \amp = 13\\ 2\sqrt{x} + 1 - 1\amp = 13 - 1\\ 2\sqrt{x} \amp = 12\\ \frac{2sqrt{x}}{2} \amp = \frac{12}{2}\\ \sqrt{x} \amp = 6 \end{align*}
    To undo the square root, we square both sides:
    \begin{align*} \left(\sqrt{x}\right)^2 \amp = 6^2\\ x \amp 36 \end{align*}
    🔗
    \(x\)
    🔗
    🔗

    Check for Understanding 2.5.8.

    Solve \(\sqrt{2x} + 1 = 13\) and \(2\sqrt{x+1} = 13\) for \(x\text{.}\)
    🔗
    Solution.
    \begin{align*} \sqrt{2x} + 1 \amp = 13\\ \sqrt{2x} + 1 - 1 \amp = 13 - 1\\ \sqrt{2x} \amp = 12\\ \left(\sqrt{2x}\right)^2 \amp = 12^2\\ 2x \amp = 144\\ \frac{2x}{2} \amp = \frac{144}{2}\\ x \amp = 72 \end{align*}
    🔗
    \begin{align*} 2\sqrt{x+1} \amp = 13\\ \frac{2\sqrt{x+1}}{2} \amp = \frac{13}{2}\\ \sqrt{x+1} \amp = 6.5\\ \left(\sqrt{x+1}\right)^2 \amp = (6.5)^2\\ x+1 \amp = 42.5\\ x + 1 - 1 \amp = 42.5 - 1\\ x \amp = 41.5 \end{align*}
    🔗
    🔗
